# 🎈 أصدقاء التعلم | Learning Friends

تطبيق Flutter تعليمي تفاعلي للأطفال (6-9 سنوات) — المرحلة 1: الحروف، الأرقام، الألوان، الأشكال.
ثنائي اللغة (عربي/إنجليزي)، مع نطق صوتي عند الضغط على أي عنصر، ولوحة متابعة لأولياء الأمور.

## ✨ المزايا المُنفَّذة (المرحلة 1)

- 🔤 محور الحروف (28 حرف عربي / 26 حرف إنجليزي حسب اللغة المختارة)
- 🔢 محور الأرقام (1-10)
- 🎨 محور الألوان (8 ألوان أساسية)
- 🔺 محور الأشكال (6 أشكال)
- 🔊 **نطق بالضغط فقط** — أي عنصر بالتطبيق يُنطق صوته فقط عند لمسه، أبداً تلقائياً
- ⭐ نظام نجوم ومكافآت بدون ضغط أو خسارة سلبية
- 👨‍👩‍👧 لوحة والدين محمية بسؤال حسابي بسيط، تعرض تقدم الطفل التفصيلي
- 💾 حفظ التقدم محلياً بالكامل (بدون إنترنت أو حسابات)

## ⚠️ ملاحظة صادقة مهمة: النطق الصوتي

يستخدم التطبيق حالياً **Text-to-Speech (TTS)** المدمج بنظام التشغيل (`flutter_tts`) بدل تسجيلات صوتية حقيقية. هذا قرار عملي للمرحلة الأولى لأنه:
- لا يحتاج ملفات صوت خارجية (يبقي حجم التطبيق صغيراً)
- يعمل أوفلاين تلقائياً على أي جهاز

**لكن** له حدود يجب معرفتها: محرك TTS العام ينطق **اسم** الحرف الأقرب لما هو متاح آلياً، وليس بالضرورة **الصوت الصوتي المنفصل الدقيق** (الصوتيات / Phonics — مثل الفرق بين نطق "بِ" الصوتي و"الباء" كاسم). لتفعيل صوتيات دقيقة تربوياً بمستوى احترافي كامل لاحقاً، الخطوة التالية المقترحة هي **تسجيل أصوات بشرية حقيقية** لكل حرف بصوته الصحيح (هذا مذكور بخارطة الطريق أدناه).

## 📱 كيف تحصل على ملف APK حقيقي (بدون Flutter على جهازك)

مجلد `android/` مبني **يدوياً بالكامل** ومرفوع ضمن المشروع (14 ملف: إعدادات Gradle، AndroidManifest، MainActivity بلغة Kotlin، أيقونات التطبيق بكل الأحجام...). الاستثناء الوحيد هو ملف واحد فقط: `gradle-wrapper.jar` — وهو ملف Java مُصرَّف (binary) يستحيل كتابته كنص. خطوة معزولة تماماً بملف البناء تولّده هو فقط، ولا تلمس أي ملف آخر بمشروعك.

بما أن جهازك لا يدعم تشغيل Flutter SDK محلياً، استخدم **GitHub Actions** (بناء سحابي مجاني تلقائي):

### الخطوات:
1. أنشئ مستودع (Repository) جديد على GitHub
2. ارفع محتوى هذا المجلد بالكامل إليه (تأكد إن `.github` مرفوع أيضاً — أحياناً يكون مخفي، فعّل إظهار الملفات المخفية عند الرفع)
3. روح لتبويب **Actions** أعلى صفحة المستودع على GitHub
4. بيبدأ بناء APK تلقائياً خلال دقائق (تقدر تشوف التقدم مباشرة)
5. بعد ما يخلص (علامة ✓ خضراء)، اضغط على البناء (Build) → بتلقى ملف باسم **kids-learning-app-apk** بأسفل الصفحة تحت قسم **Artifacts**
6. حمّله — هذا ملف APK حقيقي جاهز للتثبيت على أي هاتف أندرويد

### ملاحظة
كل مرة ترفع تعديل جديد على الكود (git push)، ينبني APK جديد تلقائياً بدون أي تدخل يدوي.
